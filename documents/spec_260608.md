## 3. Module Specifications

### 3.1 Global Router Unit

**파일**: `src/grt/model/router.py`

#### 역할

입력 세그먼트 $X_t \in \mathbb{R}^{B \times N \times D}$와 현재 레지스터 상태 $S_t \in \mathbb{R}^{B \times M \times D}$를 받아, 각 레지스터에 대한 읽기 게이트 $R_t$와 쓰기 게이트 $W_t$를 출력한다.

#### 인터페이스

```python
class GlobalRouterUnit(nn.Module):
    def __init__(self, cfg: RouterConfig) -> None: ...

    def forward(
        self,
        x: Tensor,          # [B, N, D]  입력 세그먼트
        s: Tensor,          # [B, M, D]  현재 레지스터 상태
    ) -> tuple[Tensor, Tensor, Tensor]:
        """
        Returns:
            r_gate : Tensor  [B, M, 1]  읽기 게이트 (0~1)
            w_gate : Tensor  [B, M, 1]  쓰기 게이트 (0~1)
            attn_w : Tensor  [B, N]     PMA 어텐션 가중치 (RTLA Panel C용)
        """
```

#### 내부 서브모듈

| 서브모듈            | 클래스                  | 출력 shape  | 설명                         |
| ------------------- | ----------------------- | ----------- | ---------------------------- |
| Attention Pooling   | `AttentionPool`         | `[B, 1, D]` | learnable query → X_t 어텐션 |
| Register Cross-Attn | `nn.MultiheadAttention` | `[B, 1, D]` | X_rich × S_t                 |
| Gate MLP (Read)     | `GateMLP`               | `[B, M, 1]` | Sigmoid 출력                 |
| Gate MLP (Write)    | `GateMLP`               | `[B, M, 1]` | Sigmoid 출력, bias=-2.0      |

#### `AttentionPool` 상세

```
입력: X ∈ [B, N, D]
q_pool: nn.Parameter([1, 1, D]), 초기화: xavier_uniform

K = X @ W_K    [B, N, D]
V = X @ W_V    [B, N, D]
alpha = softmax(q_pool @ K^T / sqrt(D))   [B, 1, N]
X_rich = alpha @ V                         [B, 1, D]

반환: (X_rich, alpha.squeeze(1))  → alpha shape: [B, N]
```

#### `GateMLP` 상세

```
입력: G ∈ [B, 1, 2D]  (브로드캐스트하여 M개 슬롯에 적용)
구조: Linear(2D → hidden) → GELU → Linear(hidden → M)
출력 전처리: reshape to [B, M, 1]
최종 출력: Sigmoid(logit + bias)
```

**Write gate bias 초기화**:

```python
nn.init.constant_(self.mlp_w.layers[-1].bias, cfg.write_gate_bias_init)
# cfg.write_gate_bias_init = -2.0
```

#### Register Dropout (학습 시 전용)

```python
# 쓰기 게이트 로짓 마스킹
if self.training and cfg.dropout_prob > 0:
    mask = torch.bernoulli(
        torch.full((B, M, 1), cfg.dropout_prob, device=logit_w.device)
    ).bool()
    logit_w = logit_w.masked_fill(mask, float('-inf'))
w_gate = torch.sigmoid(logit_w)
```

#### `mode: "mean_pool"` 어블레이션

`router.mode == "mean_pool"` 일 때 Step 1을 다음으로 대체:

```python
X_rich = x.mean(dim=1, keepdim=True)   # [B, 1, D]
attn_w = torch.ones(B, N, device=x.device) / N   # uniform (dummy)
```

Step 2 (register cross-attn)는 동일하게 유지.

### 3.2 ALU (Transformer Core)

**파일**: `src/grt/model/alu.py`

#### 역할

마스킹된 레지스터 $\tilde{S}_t$와 입력 토큰 $X_t$를 연결(concatenate)하여 항상 고정 길이 $(N+M)$ 시퀀스를 처리하고, 토큰 출력 $Y_t$와 레지스터 후보 $\Delta S_t$를 분리 반환한다.

#### 인터페이스

```python
class ALU(nn.Module):
    def __init__(self, cfg: ALUConfig) -> None: ...

    def forward(
        self,
        x: Tensor,          # [B, N, D]  입력 토큰
        s_masked: Tensor,   # [B, M, D]  마스킹된 레지스터
    ) -> tuple[Tensor, Tensor]:
        """
        Returns:
            y_tokens   : Tensor  [B, N, D]  토큰 출력
            delta_s    : Tensor  [B, M, D]  레지스터 쓰기 후보
        """
```

#### 내부 구조

```
seq = concat([x, s_masked], dim=1)     # [B, N+M, D]
out = TransformerEncoder(seq)          # [B, N+M, D]
y_tokens = out[:, :N, :]              # [B, N, D]
delta_s  = out[:, N:, :]              # [B, M, D]
```

**주의**: ALU 내부 self-attention은 인과적(causal) 마스킹을 적용하지 않는다. 세그먼트 전체가 하나의 고정 단위로 처리된다. 언어 모델링 시 causal mask는 세그먼트 내 토큰 위치에만 선택적으로 적용 가능하다 (실험 설정에 따라 결정).

#### `TransformerEncoder` 구성

```python
encoder_layer = nn.TransformerEncoderLayer(
    d_model=cfg.d_model,
    nhead=cfg.nhead,
    dim_feedforward=cfg.d_ff,
    dropout=cfg.dropout,
    activation=cfg.activation,
    batch_first=True,
    norm_first=True,          # Pre-LN (학습 안정성)
)
self.encoder = nn.TransformerEncoder(encoder_layer, num_layers=cfg.num_layers)
```

### 3.3 Register Write-back

**파일**: `src/grt/model/writeback.py`

#### 역할

쓰기 게이트 $W_t$를 이용하여 현재 레지스터 $S_t$를 업데이트한다.

#### 인터페이스

```python
class RegisterWriteback(nn.Module):
    """파라미터 없음. 순수 게이트 연산."""

    def forward(
        self,
        s: Tensor,          # [B, M, D]  현재 레지스터 상태 S_t
        delta_s: Tensor,    # [B, M, D]  ALU 출력 ΔS_t
        w_gate: Tensor,     # [B, M, 1]  쓰기 게이트 W_t
    ) -> tuple[Tensor, Tensor]:
        """
        Returns:
            s_next      : Tensor  [B, M, D]  업데이트된 레지스터 S_{t+1}
            pred_error  : Tensor  [B, M]     예측 오차 ||ΔS_t - S_t||_2 (RTLA Panel D용)
        """
```

#### 연산

```python
s_next = (1.0 - w_gate) * s + w_gate * delta_s
pred_error = torch.norm(delta_s - s, dim=-1)   # [B, M]
return s_next, pred_error
```

**Jacobian 검증 주석**: `∂S_{t+1}/∂S_t = diag(1 - W_t)`. 각 슬롯 $i$에 대해 $W_{t,i} \to 0$ 이면 gradient highway, $W_{t,i} \to 1$ 이면 gradient cut-off.

### 3.4 GRT Model (Top-level)

**파일**: `src/grt/model/grt.py`

#### 역할

토크나이즈된 시퀀스를 세그먼트 단위로 분할하여 처리하는 최상위 모델. 레지스터 상태를 세그먼트 간에 전달하며, 언어 모델링 헤드까지 포함한다.

#### 인터페이스

```python
class GRTModel(nn.Module):
    def __init__(self, cfg: GRTConfig) -> None: ...

    def forward(
        self,
        input_ids: Tensor,               # [B, L]  토큰 ID (L = 전체 시퀀스 길이)
        return_trace: bool = False,      # RTLA 학습 시 trace 반환 여부
    ) -> GRTOutput:
        """
        Returns:
            GRTOutput(
                logits     : Tensor  [B, L, V]
                loss       : Tensor  scalar  (if labels provided)
                trace      : TraceBuffer | None
            )
        """

    def init_registers(self, batch_size: int) -> Tensor:
        """S_0 반환: [B, M, D]. s0_learnable=True이면 학습된 파라미터 확장."""
```

#### Forward Pass 세부 흐름

```python
# 1. 임베딩
token_emb = self.embedding(input_ids)                  # [B, L, D]

# 2. 세그먼트 분할
segments = token_emb.split(cfg.segment_len, dim=1)     # List of [B, N, D]

# 3. 레지스터 초기화
s = self.init_registers(B)                             # [B, M, D]

all_logits = []
for t, x_t in enumerate(segments):
    # Stage 1: 라우팅
    r_gate, w_gate, attn_w = self.router(x_t, s)

    # Stage 2: 조건부 읽기
    s_masked = r_gate * s                              # [B, M, D]

    # Stage 3: ALU 실행
    y_t, delta_s = self.alu(x_t, s_masked)

    # Stage 4: Write-back
    s, pred_error = self.writeback(s, delta_s, w_gate)

    # LM 헤드
    logits_t = self.lm_head(y_t)                      # [B, N, V]
    all_logits.append(logits_t)

    # RTLA 버퍼에 기록 (return_trace=True 시)
    if return_trace:
        trace.record(t, r_gate, w_gate, s, pred_error, attn_w)

logits = torch.cat(all_logits, dim=1)                  # [B, L, V]
```

#### `GRTOutput` 데이터클래스

```python
@dataclass
class GRTOutput:
    logits: Tensor                      # [B, L, V]
    loss: Tensor | None = None          # cross-entropy loss (labels 제공 시)
    trace: "TraceBuffer | None" = None  # RTLA 데이터
```

#### `S_0` 초기화

```python
# s0_learnable = True
self.s0 = nn.Parameter(torch.zeros(1, cfg.num_registers, cfg.d_model))

def init_registers(self, batch_size: int) -> Tensor:
    if self.cfg.register.s0_learnable:
        return self.s0.expand(batch_size, -1, -1).clone()
    else:
        return torch.zeros(batch_size, self.cfg.num_registers, self.cfg.d_model)
```

**주의**: `.clone()` 필수. `expand()`는 메모리를 공유하므로 in-place 업데이트 시 그래디언트 오류 발생.

## 4. Data Flow Contract

세그먼트 $t$에서의 전체 텐서 흐름:

```
input_ids [B, L]
    │ embedding
    ▼
X_t       [B, N, D]       ─────────────────────────────┐
S_t       [B, M, D]  ──► router ──► R_t [B, M, 1]      │
                               └──► W_t [B, M, 1]      │
                               └──► α_t [B, N]         │ (RTLA)
                                                        │
R_t ⊙ S_t → S̃_t [B, M, D]                             │
                                                        │
[X_t ; S̃_t] [B, N+M, D] ──► ALU ──► Y_t [B, N, D]     │
                                └──► ΔS_t [B, M, D]    │
                                                        │
(1-W_t)⊙S_t + W_t⊙ΔS_t → S_{t+1} [B, M, D]           │
‖ΔS_t - S_t‖₂            → err_t  [B, M]    ──────────┘ (RTLA)
                                                        │
Y_t ──► lm_head ──► logits_t [B, N, V]
```

**불변 조건**:

- ALU 입력 시퀀스 길이는 항상 `N + M = 160` (세그먼트 길이와 무관)
- 게이트 텐서 $R_t, W_t$의 값 범위: `[0, 1]` (sigmoid 출력)
- 레지스터 상태 $S_t$는 세그먼트 경계에서 `detach()` 없이 전파 (BPTT)

## 5. Register Trace Log Analyzer (RTLA)

**파일**: `src/grt/rtla/analyzer.py`, `src/grt/rtla/plots.py`

### 5.1 `TraceBuffer`

세그먼트별로 수집된 게이팅 신호를 메모리에 누적하고 `.npz`로 직렬화한다.

```python
@dataclass
class TraceBuffer:
    r_gates:     list[Tensor]   # 각 원소: [B, M, 1]
    w_gates:     list[Tensor]   # 각 원소: [B, M, 1]
    s_norms:     list[Tensor]   # 각 원소: [B, M]  (‖S_t‖₂)
    pred_errors: list[Tensor]   # 각 원소: [B, M]
    attn_weights:list[Tensor]   # 각 원소: [B, N]

    def to_numpy(self, batch_idx: int = 0) -> dict[str, np.ndarray]:
        """batch_idx번째 샘플을 [T, M] / [T, N] numpy 배열로 변환."""
        return {
            "r_gates":      _stack(self.r_gates,      batch_idx),  # [T, M]
            "w_gates":      _stack(self.w_gates,      batch_idx),  # [T, M]
            "s_norms":      _stack(self.s_norms,      batch_idx),  # [T, M]
            "pred_errors":  _stack(self.pred_errors,  batch_idx),  # [T, M]
            "attn_weights": _stack(self.attn_weights, batch_idx),  # [T, N]
        }

    def save(self, path: str, batch_idx: int = 0) -> None:
        """로컬 .npz 백업."""
        np.savez_compressed(path, **self.to_numpy(batch_idx))

    def gate_stats(self, batch_idx: int = 0) -> dict[str, np.ndarray]:
        """
        W&B Table 업로드용 슬롯별 요약 통계 계산.

        Returns dict with keys:
            slot_id        : [M]   슬롯 인덱스
            mean_w         : [M]   평균 쓰기 게이트
            mean_r         : [M]   평균 읽기 게이트
            mean_norm      : [M]   평균 레지스터 norm
            mean_pred_err  : [M]   평균 예측 오차
            tier           : [M]   추정 역할 ("long-term" | "working" | "scratch")
        """
        arrays = self.to_numpy(batch_idx)
        mean_w = arrays["w_gates"].mean(axis=0)           # [M]
        mean_r = arrays["r_gates"].mean(axis=0)           # [M]
        mean_norm = arrays["s_norms"].mean(axis=0)        # [M]
        mean_err = arrays["pred_errors"].mean(axis=0)     # [M]
        tier = np.where(mean_w < 0.2, "long-term",
               np.where(mean_w < 0.7, "working", "scratch"))
        return {
            "slot_id":       np.arange(len(mean_w)),
            "mean_w":        mean_w,
            "mean_r":        mean_r,
            "mean_norm":     mean_norm,
            "mean_pred_err": mean_err,
            "tier":          tier,
        }
```

class RegisterAnalyzer:
    def __init__(self, model: GRTModel) -> None:
        self.model = model

    def run_trace(
        self,
        input_ids: Tensor,       # [1, L]  단일 시퀀스
    ) -> TraceBuffer:
        """model.forward(return_trace=True) 호출 후 TraceBuffer 반환."""
        with torch.no_grad():
            out = self.model(input_ids, return_trace=True)
        return out.trace

    def save_trace(self, trace: TraceBuffer, path: str) -> None:
        """로컬 .npz 백업 저장."""

### 6.1 학습 루프 의사코드

```python
# W&B run은 Trainer.__init__에서 초기화됨 (Section 11.2 참고)
for step, batch in enumerate(dataloader):
    input_ids = batch["input_ids"]    # [B, L]
    labels    = batch["labels"]       # [B, L]  (input_ids shifted by 1)

    do_trace = (step % cfg.rtla.trace_every_n_steps == 0)

    with autocast(dtype=torch.bfloat16):
        out = model(input_ids, return_trace=do_trace)
        loss = F.cross_entropy(
            out.logits.view(-1, cfg.model.vocab_size),
            labels.view(-1),
            ignore_index=-100,
        )

    loss.backward()
    grad_norm = nn.utils.clip_grad_norm_(model.parameters(), cfg.training.grad_clip)
    optimizer.step()
    scheduler.step()
    optimizer.zero_grad()

    # ── W&B 메트릭 로깅 ──────────────────────────────────────────
    if step % cfg.wandb.log_freq == 0:
        wandb_logger.log_train_step(
            step=step,
            loss=loss.item(),
            lr=scheduler.get_last_lr()[0],
            grad_norm=grad_norm.item(),
        )

    # ── RTLA trace 저장 및 W&B 업로드 ───────────────────────────
    if do_trace:
        trace_path = f"{cfg.rtla.output_dir}/step_{step:06d}.npz"
        analyzer.save_trace(out.trace, trace_path)
        if cfg.rtla.upload_panels:
            rtla_uploader.upload(out.trace, step=step, trace_path=trace_path)

    # ── 평가 ────────────────────────────────────────────────────
    if step % cfg.logging.eval_every_n_steps == 0:
        eval_metrics = evaluate(model, eval_loader, cfg)
        wandb_logger.log_eval(step=step, **eval_metrics)

    # ── 체크포인트 저장 ─────────────────────────────────────────
    if step % cfg.logging.save_every_n_steps == 0:
        ckpt_path = f"{cfg.logging.save_dir}/step_{step:06d}.pt"
        save_checkpoint(model, optimizer, scheduler, step, ckpt_path)
        if cfg.wandb.checkpoint_artifact:
            wandb_logger.log_checkpoint(ckpt_path, step=step)
```

```python
import wandb

class WandbLogger:
    def __init__(self, cfg: WandbConfig, full_cfg: GRTConfig) -> None:
        """
        W&B run 초기화. wandb.enabled=False이면 no-op으로 동작.
        full_cfg 전체를 wandb.config에 저장하여 하이퍼파라미터 추적.
        """
        self.enabled = cfg.enabled
        if not self.enabled:
            return

        self.run = wandb.init(
            project=cfg.project,
            entity=cfg.entity or None,
            name=cfg.run_name or None,
            tags=cfg.tags or [],
            notes=cfg.notes or "",
            config=dataclasses.asdict(full_cfg),   # 전체 config 기록
            save_code=cfg.save_code,
        )

    def log_train_step(
        self,
        step: int,
        loss: float,
        lr: float,
        grad_norm: float,
    ) -> None:
        if not self.enabled:
            return
        wandb.log({
            "train/loss":      loss,
            "train/ppl":       math.exp(loss),
            "train/lr":        lr,
            "train/grad_norm": grad_norm,
        }, step=step)

    def log_eval(
        self,
        step: int,
        loss: float,
        tokens_per_sec: float,
        peak_mem_gb: float,
    ) -> None:
        if not self.enabled:
            return
        wandb.log({
            "eval/loss":           loss,
            "eval/ppl":            math.exp(loss),
            "eval/tokens_per_sec": tokens_per_sec,
            "eval/peak_mem_gb":    peak_mem_gb,
        }, step=step)

    def log_checkpoint(self, ckpt_path: str, step: int) -> None:
        """체크포인트 파일을 W&B Artifact로 업로드."""
        if not self.enabled:
            return
        artifact = wandb.Artifact(
            name=f"checkpoint-step-{step}",
            type="model",
            metadata={"step": step},
        )
        artifact.add_file(ckpt_path)
        wandb.log_artifact(artifact)

    def alert(self, title: str, text: str, level: str = "warn") -> None:
        if not self.enabled:
            return
        wandb.alert(title=title, text=text, level=level)

    def finish(self) -> None:
        if self.enabled:
            wandb.finish()
```

### 11.3 `RTLAUploader`

```python
import wandb
import numpy as np

class RTLAUploader:
    def __init__(self, analyzer: RegisterAnalyzer, cfg: RTLAConfig) -> None:
        self.analyzer = analyzer
        self.cfg = cfg

    def upload(
        self,
        trace: TraceBuffer,
        step: int,
        trace_path: str,
    ) -> None:
        """
        1. 4-panel Figure → wandb.Image 업로드
        2. gate_stats → wandb.Table 업로드
        3. 요약 스칼라 업로드
        """
        arrays = trace.to_numpy(batch_idx=0)
        stats  = trace.gate_stats(batch_idx=0)

        log_dict: dict = {}

        # ── 패널 이미지 ────────────────────────────────────────
        if self.cfg.upload_panels:
            fig = make_figure(arrays)          # plots.py
            log_dict["rtla/panels"] = wandb.Image(
                fig,
                caption=f"RTLA panels @ step {step}",
            )
            plt.close(fig)

        # ── 슬롯 통계 Table ────────────────────────────────────
        if self.cfg.upload_table:
            table = wandb.Table(
                columns=["slot_id", "mean_w", "mean_r",
                         "mean_norm", "mean_pred_err", "tier"],
            )
            M = len(stats["slot_id"])
            for i in range(M):
                table.add_data(
                    int(stats["slot_id"][i]),
                    float(stats["mean_w"][i]),
                    float(stats["mean_r"][i]),
                    float(stats["mean_norm"][i]),
                    float(stats["mean_pred_err"][i]),
                    str(stats["tier"][i]),
                )
            log_dict["rtla/gate_stats"] = table

        # ── 요약 스칼라 ────────────────────────────────────────
        w_gates = arrays["w_gates"]                   # [T, M]
        log_dict.update({
            "rtla/mean_w":               float(w_gates.mean()),
            "rtla/mean_r":               float(arrays["r_gates"].mean()),
            "rtla/w_std_across_slots":   float(w_gates.mean(axis=0).std()),
            "rtla/dead_register_count":  int((w_gates.mean(axis=0) < 0.05).sum()),
        })

        wandb.log(log_dict, step=step)
```

### 12.4 `config.py` 구현 명세

YAML을 Python dataclass로 변환한다. 외부 라이브러리(Hydra 등) 없이 `pyyaml`만 사용한다.

```python
# src/grt/config.py
from __future__ import annotations
from dataclasses import dataclass, field
import yaml

@dataclass
class ALUConfig:
    num_layers: int = 2
    nhead: int = 4
    d_ff: int = 1024
    dropout: float = 0.1
    activation: str = "gelu"

@dataclass
class RouterConfig:
    mode: str = "cross_attn"
    pool_nhead: int = 4
    mlp_hidden: int = 512

@dataclass
class RegisterConfig:
    write_gate_bias_init: float = -2.0
    dropout_prob: float = 0.10
    s0_learnable: bool = True

@dataclass
class ModelConfig:
    segment_len: int = 128
    num_registers: int = 32
    d_model: int = 256
    vocab_size: int = 32000
    alu: ALUConfig = field(default_factory=ALUConfig)
    router: RouterConfig = field(default_factory=RouterConfig)
    register: RegisterConfig = field(default_factory=RegisterConfig)

@dataclass
class WandbConfig:
    enabled: bool = False
    project: str = "grt-research"
    entity: str | None = None
    run_name: str | None = None
    tags: list[str] = field(default_factory=list)
    notes: str = ""
    log_freq: int = 50
    watch_model: bool = False
    save_code: bool = True
    checkpoint_artifact: bool = True

@dataclass
class GRTConfig:
    model: ModelConfig = field(default_factory=ModelConfig)
    wandb: WandbConfig = field(default_factory=WandbConfig)

def load_config(path: str) -> GRTConfig:
    """YAML 파일을 GRTConfig dataclass로 변환."""
    with open(path) as f:
        raw = yaml.safe_load(f)
    return _build(GRTConfig, raw)

def _build(cls, data: dict):
    """재귀적으로 dataclass를 nested dict에서 생성."""
    import dataclasses
    if not dataclasses.is_dataclass(cls) or data is None:
        return data
    kwargs = {}
    hints = {f.name: f for f in dataclasses.fields(cls)}
    for name, field_obj in hints.items():
        if name not in data:
            kwargs[name] = field_obj.default_factory() if callable(field_obj.default_factory) else field_obj.default
        else:
            sub_cls = field_obj.type if isinstance(field_obj.type, type) else None
            if sub_cls and dataclasses.is_dataclass(sub_cls):
                kwargs[name] = _build(sub_cls, data[name])
            else:
                kwargs[name] = data[name]
    return cls(**kwargs)
```

### 12.5 Sanity Check 스크립트

**파일**: `scripts/sanity_check.py`

이 스크립트가 **모두 PASS**를 출력하면 Phase 1 완료다. Kaggle 노트북에서 `%run scripts/sanity_check.py` 로 실행한다.

```python
# scripts/sanity_check.py
"""
GRT Phase 1 Sanity Check
모든 CHECK가 PASS되면 Phase 2로 진행한다.
"""
import sys, os
sys.path.insert(0, os.path.join(os.path.dirname(__file__), "..", "src"))

import torch
import traceback

from grt.config import load_config, GRTConfig
from grt.model.grt import GRTModel

PASS = "✅ PASS"
FAIL = "❌ FAIL"

def run(name: str, fn):
    try:
        fn()
        print(f"{PASS}  {name}")
        return True
    except Exception as e:
        print(f"{FAIL}  {name}")
        traceback.print_exc()
        return False

# ── 공통 픽스처 ────────────────────────────────────────────────────
cfg = GRTConfig()    # 기본값으로 생성 (YAML 없이도 동작 확인)
B, N, M, D, V = 2, cfg.model.segment_len, cfg.model.num_registers, cfg.model.d_model, cfg.model.vocab_size
L = N * 4            # 4개 세그먼트
device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
model = GRTModel(cfg.model).to(device)
model.eval()

results = []

# ── CHECK 1: 파라미터 수 ────────────────────────────────────────────
def check_param_count():
    n = sum(p.numel() for p in model.parameters())
    assert n > 0, "모델 파라미터가 없음"
    print(f"      파라미터 수: {n:,}")

results.append(run("파라미터 수 > 0", check_param_count))

# ── CHECK 2: Forward pass shape ─────────────────────────────────────
def check_forward_shape():
    ids = torch.randint(0, V, (B, L), device=device)
    with torch.no_grad():
        out = model(ids)
    assert out.logits.shape == (B, L, V), \
        f"logits shape 오류: expected {(B, L, V)}, got {out.logits.shape}"

results.append(run("Forward pass output shape [B, L, V]", check_forward_shape))

# ── CHECK 3: ALU 입력 길이 고정 N+M ────────────────────────────────
def check_alu_fixed_len():
    # ALU에 hook을 걸어 실제 입력 shape 확인
    observed = []
    def hook(module, inp, out):
        observed.append(inp[0].shape[1])   # seq_len 차원
    h = model.alu.encoder.layers[0].register_forward_hook(hook)
    ids = torch.randint(0, V, (B, L), device=device)
    with torch.no_grad():
        model(ids)
    h.remove()
    expected = N + M
    for seq_len in observed:
        assert seq_len == expected, \
            f"ALU seq_len 오류: expected {expected}, got {seq_len}"

results.append(run(f"ALU 입력 길이 항상 N+M={N+M}", check_alu_fixed_len))

# ── CHECK 4: 게이트 범위 [0, 1] ─────────────────────────────────────
def check_gate_range():
    ids = torch.randint(0, V, (B, L), device=device)
    with torch.no_grad():
        out = model(ids, return_trace=True)
    trace = out.trace
    for t, (r, w) in enumerate(zip(trace.r_gates, trace.w_gates)):
        assert r.min() >= 0 and r.max() <= 1, f"R_gate out of [0,1] at t={t}"
        assert w.min() >= 0 and w.max() <= 1, f"W_gate out of [0,1] at t={t}"

results.append(run("게이트 값 범위 [0, 1]", check_gate_range))

# ── CHECK 5: 레지스터가 세그먼트마다 변경됨 ─────────────────────────
def check_register_updates():
    ids = torch.randint(0, V, (B, L), device=device)
    with torch.no_grad():
        out = model(ids, return_trace=True)
    trace = out.trace
    norms = [s.detach().cpu() for s in trace.s_norms]
    # 적어도 하나의 세그먼트 전환에서 norm이 변해야 함
    changes = sum(
        1 for i in range(1, len(norms))
        if not torch.allclose(norms[i], norms[i-1], atol=1e-6)
    )
    assert changes > 0, "레지스터 상태가 전혀 변하지 않음 (write gate 고장 의심)"

results.append(run("레지스터 상태가 세그먼트 간 변경됨", check_register_updates))

# ── CHECK 6: write gate 초기값 ≈ 0.12 ────────────────────────────
def check_write_gate_init():
    ids = torch.randint(0, V, (B, L), device=device)
    with torch.no_grad():
        out = model(ids, return_trace=True)
    w_mean = torch.stack(out.trace.w_gates).mean().item()
    # sigmoid(-2) ≈ 0.119; 학습 전이므로 0.05 ~ 0.30 범위면 정상
    assert 0.05 < w_mean < 0.30, \
        f"write gate 초기 평균={w_mean:.3f}, bias 초기화 확인 필요"
    print(f"      write gate 초기 평균: {w_mean:.4f}")

results.append(run("Write gate 초기 평균 ≈ 0.12 (0.05~0.30)", check_write_gate_init))

# ── CHECK 7: Gradient flow (loss.backward 오류 없음) ────────────────
def check_backward():
    model.train()
    ids = torch.randint(0, V, (B, L), device=device)
    out = model(ids)
    labels = ids.clone()
    import torch.nn.functional as F
    loss = F.cross_entropy(out.logits.view(-1, V), labels.view(-1))
    loss.backward()
    # S_0가 learnable이면 gradient 있어야 함
    if cfg.model.register.s0_learnable:
        assert model.s0.grad is not None, "S_0 gradient가 None"
    model.eval()

results.append(run("Backward pass 오류 없음 + S_0 gradient 전파", check_backward))

# ── CHECK 8: VRAM 사용량 (T4 16GB 기준) ────────────────────────────
def check_vram():
    if not torch.cuda.is_available():
        print("      GPU 없음, VRAM 체크 스킵")
        return
    torch.cuda.reset_peak_memory_stats(device)
    ids = torch.randint(0, V, (B, L), device=device)
    with torch.no_grad():
        model(ids)
    peak_gb = torch.cuda.max_memory_allocated(device) / 1e9
    assert peak_gb < 4.0, f"VRAM {peak_gb:.2f} GB > 4 GB 초과 (단일 forward, B=2)"
    print(f"      Peak VRAM: {peak_gb:.3f} GB")

results.append(run("VRAM 사용량 < 4 GB (B=2, L=512)", check_vram))

# ── 결과 요약 ──────────────────────────────────────────────────────
print("\n" + "─" * 50)
passed = sum(results)
total  = len(results)
print(f"결과: {passed}/{total} PASS")
if passed == total:
    print("🎉 모든 체크 통과 — Phase 2로 진행하세요.")
else:
    print("⚠️  실패 항목을 수정한 후 다시 실행하세요.")
    sys.exit(1)
```
